# ☕ Cafe Profile Template

> Sebuah template website profil cafe yang elegan, modern, dan siap pakai untuk UMKM.

Template ini dibuat khusus untuk membantu UMKM kuliner memiliki tampilan online yang profesional. Dibangun dengan pendekatan **simple namun tetap elegan** — fokus pada UI yang rapi, cepat diakses, dan nyaman dilihat dari HP.

---

## ✨ Fitur Utama

| Fitur | Keterangan |
|-------|------------|
| 📱 **Responsive Design** | Tampilan optimal di desktop, tablet, dan HP |
| 🎨 **Desain Modern & Elegan** | Warna emas, hitam, putih — cocok untuk cafe kekinian |
| 🚀 **Fast Loading** | Ringan, tanpa framework berat |
| 📍 **Google Maps Terintegrasi** | Memudahkan pelanggan menemukan lokasi |
| 💬 **Tombol WhatsApp Aktif** | Order langsung via WhatsApp dengan satu klik |
| 📝 **Form Kontak Sederhana** | Pelanggan bisa kirim pesan langsung dari website |

## 🚀 Cara Menggunakan

1. **Clone repository ini**
   ```bash
   git clone https://github.com/ryygans/frontend-project.git
```

1. Buka file index.html di browser
2. Sesuaikan konten (nama cafe, menu, alamat, nomor WA, foto) sesuai kebutuhan
3. Deploy ke hosting gratis seperti Netlify atau Vercel
