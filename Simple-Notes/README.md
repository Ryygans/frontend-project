# 📝 Smart Notes - Aplikasi Catatan Modern

Aplikasi catatan digital yang sederhana namun powerful untuk mencatat ide, tugas, dan kegiatan sehari-hari. Dilengkapi dengan fitur lengkap dan penyimpanan lokal yang membuat catatanmu tetap aman meskipun browser ditutup.

## ✨ Fitur Unggulan

- ✅ **Tambah Catatan** - Form dengan judul dan isi catatan
- 📋 **Lihat Catatan** - Tampilan card yang rapi dengan cuplikan isi
- ✏️ **Edit Catatan** - Ubah catatan kapan saja dengan mudah
- 🗑️ **Hapus Catatan** - Dilengkapi konfirmasi sebelum menghapus
- 🎯 **Tandai Selesai** - Fitur checkbox untuk menandai catatan yang sudah dikerjakan
- 💾 **Penyimpanan Lokal** - Data tersimpan di localStorage browser
- 📱 **Responsif** - Nyaman digunakan di desktop, tablet, maupun smartphone
- 🎨 **Desain Modern** - Tampilan gradient yang menarik dan animasi halus

## 🚀 Demo Langsung

Kunjungi demo aplikasi: [Smart Notes Demo](https://simplenotesmart.netlify.app)

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur halaman
- **CSS3** - Styling dan animasi (Flexbox, Grid, Media Queries)
- **JavaScript (Vanilla)** - Logika aplikasi tanpa framework
- **LocalStorage API** - Penyimpanan data di browser

## 💻 Cara Menjalankan

1. **Clone repository ini**
   ```bash
   git clone https://github.com/Ryygans/frontend-project/Simple-Notes.git
```

1. Masuk ke folder project
   ```bash
   cd Simple-Notes
   ```
2. Buka file index.html
   · Double click file index.html
   · Atau gunakan Live Server di VS Code
3. Mulai membuat catatan! 🎉

📖 Cara Penggunaan

Menambah Catatan

1. Isi judul catatan pada kolom "Judul Catatan"
2. Isi detail catatan pada kolom "Isi Catatan"
3. Klik tombol "Simpan Catatan"
4. Catatan akan langsung muncul di bawah form

Mengedit Catatan

1. Pada card catatan, klik tombol "Edit"
2. Form akan terisi dengan data catatan yang dipilih
3. Ubah judul atau isi catatan sesuai keinginan
4. Klik "Simpan Perubahan"

Menandai Selesai

· Centang checkbox "Selesai" pada card catatan
· Catatan akan memiliki tampilan coretan (strikethrough)
· Centang ulang untuk membatalkan

Menghapus Catatan

· Klik tombol "Hapus" pada card catatan
· Konfirmasi penghapusan akan muncul
· Klik "OK" untuk menghapus, "Cancel" untuk membatalkan
